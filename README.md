# Docker flutter web app


## Build the docker image command
```
docker build -t <DOCKER NAME> .
```

## After Success building image
## Run the docker image
```
docker run -d -p <PORT FOR DOCKER>:<THE EXPOSE PORT> <DOCKER NAME>
```
